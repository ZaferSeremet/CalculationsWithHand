## How It Works

1. **Hand Tracking** вҖ“ MediaPipe detects your hand and tracks finger positions in real-time.
2. **Air Drawing** вҖ“ Raise your index finger to draw on a virtual canvas overlaid on the camera feed.
3. **Character Segmentation** вҖ“ The drawing is converted to binary, contours are extracted, and each character is isolated using bounding boxes.
4. **CNN Recognition** вҖ“ Each 28Г—28 segment is fed into a custom-trained CNN that classifies it as a digit (0вҖ“9) or an operator (+, вҲ’, Г—, Г·).
5. **Evaluation** вҖ“ The detected expression is evaluated and the result is displayed on screen.

Additionally, the **Gemini API** can be used as a fallback for complex or ambiguous expressions.

## Features

- вңҚпёҸ Draw math expressions using hand gestures (via webcam)
- рҹ§  Custom CNN model trained on your own handwriting
- рҹ”„ **Interactive correction mode** вҖ“ teach the model from its mistakes
  - **Merge** fragmented segments (e.g., `Г·` detected as three parts вҶ’ join into one)
  - **Split** merged characters (e.g., `97` detected as one blob вҶ’ split into two)
- рҹ”Ғ **One-key retraining** вҖ“ retrain the model on the fly with corrected data
- вҳҒпёҸ Gemini API integration for complex expressions

## Controls

| Key | Action |
|-----|--------|
| **Draw** | Raise index finger only |

| Key | Action |

   - Replace `YOUR_API_KEY_HERE` in `main.py` with your key

2. **Prepare your dataset** in the following folder structure:
   ```

   в”ңв”Җв”Җ 0/        # Images of handwritten "0"
   ```

## Self-Learning Correction System

What makes this project unique is the **interactive feedback loop**:

1. Draw an equation вҶ’ model makes predictions
2. If a character is wrong, press `d` and correct it
3. Corrected samples are saved directly to the training dataset
4. Press `r` to retrain the model with the new data
5. The model improves over time with **your specific handwriting style**

The correction mode also supports:
- **Merging** (`j`): Combine multiple segments that should be a single symbol (e.g., the dots and line of a division sign Г·)
- **Splitting** (`k`): Separate a single segment that contains multiple characters (e.g., "97" recognized as one unit)
